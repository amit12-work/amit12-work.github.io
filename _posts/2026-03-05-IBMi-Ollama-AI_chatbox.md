You will get all the information regarding the IBMi connections with LLMs. 

The idea here is to demonstrate the power of LLM using the different tools and fetching data from IBM i.

What used to happen, finance director or purchase director used to call IT manager and requests for the monthly report of the sales or give me the list of top 25 products. Then IT manager used to create ticket and get assigned to developer. Developer were using QRY400 to fetch the data and create one CSV file or write a program to do this task repepatively and handover to the respective team.

Same finance director will ask this question to AI chatbox and in certain amount of time he will get the data on his screen back or in the pdf/csv format. No need to wait to hear back from IT. Well, not good for us (I am that IT developer which used to create the report and send back to respective team). But good for the business and if business runs then and then jobs will remain, only the job format for developer will change.
